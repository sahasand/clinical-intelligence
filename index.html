Quick-Start Guide

A walkthrough of each feature and how to use it
01

Trial Search

One-shot discovery across the entire registry. The starting point for any research.

  • Combine condition + intervention + sponsor for precision
  • Click any row to open the full study detail modal
  • Status pills under the header filter the current page
  • Bookmark trials to compare them later
02

Sponsor Pipeline

See a company's entire portfolio in one view. Built for competitive intelligence.

  • Phase × Status bar shows pipeline maturity at a glance
  • Toggle Count / % for distribution view
  • Hit Compare to put two sponsors head-to-head
  • Sparkline on Total Trials shows pipeline trajectory
03

Endpoint Benchmarking

Answers "what endpoint should I use?" for a new protocol in your indication.

  • Pulls all primary endpoints across the condition
  • Smart normalization groups OS / PFS / ORR / MACE variants
  • Top-5 × Phase chart shows what's used at each stage
  • Filter by phase to focus on Phase 3 standards
04

Investigator Finder

Locate key opinion leaders and high-volume research sites for site selection.

  • Filter by condition + country / state / city
  • Top-15 PIs ranked with gold/silver/bronze badges
  • Country bar chart shows where trials concentrate
  • Export PI lists for outreach
05

Eligibility Analyzer

Understand the eligibility funnel before designing inclusion/exclusion criteria.

  • Median min/max age across all trials
  • Age distribution charts with median annotation
  • Top 20 criteria keywords across studies
  • Sex eligibility breakdown
06

Bookmarks & Export

Collect trials of interest into a session shortlist, then ship to CSV.

  • Click the ★ on any row or in the modal to save
  • Sidebar lives in the header (bookmark icon)
  • Export with sponsor/condition in filename
  • Every table has CSV export
